# Reading error messages

*Coming soon.*

This appendix will cover how to read the error messages Python
shows you when your program doesn't work. Most error messages are
trying to help. They tell you which file, which line, what the
problem is, and sometimes what to do about it.

We'll add this once the class hits Phase 3 (Python basics) and
real error messages start showing up. Until then, the short rule
is: read the bottom line of the error first. That's almost always
the most important part.
