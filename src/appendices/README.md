# Appendices

Reference material that doesn't belong inside any single
session — the kind of thing you'll come back to over and
over.

- **Getting unstuck** — the checklist for when your code
  isn't doing what you want.
- **Reading error messages** — how to make sense of the
  red text Python gives you when something goes wrong.
- **Glossary** — every programming term we've used,
  defined briefly. Grows phase by phase.
- **Keyboard shortcuts** — what every key combination on
  the class machines does.
- **Thonny reference** — quick reference for the Python
  editor we use.
- **Installing on your own computer** — for setting up
  Python and the class tools at home.

Bookmark this page. Most of these appendices stay useful
across multiple phases.
