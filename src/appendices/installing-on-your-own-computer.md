# Installing on your own computer

*Coming soon.*

Mr. Eric set up your class computer with everything you need. But
maybe you want to write Python on a laptop at home, or on a family
computer, or on a school computer that lets you install things.
This appendix will walk you through getting Python and Thonny
installed on Windows, macOS, and Linux.

For now, the short answer: install Thonny from
[thonny.org](https://thonny.org). It bundles its own copy of
Python so you don't have to install Python separately.
